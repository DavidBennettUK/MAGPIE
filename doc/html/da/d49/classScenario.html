Detailed Description

Base class for scenarios to perform GP runs.

Contains Generations of Individuals. Extend this to create concrete scenarios.

Author
David Bennett

Constructor & Destructor Documentation

Scenario::Scenario ( int  populationSize)

Constructs the scenario with a population size.

Generations will be filled with individuals until the population size. Defaults: Selection: tournament (size: 5) Max tree depth: 5 Max generations: 500 Target fitness: 100 Crossover: subtree Mutation: point Crossover rate: 0.75 Mutation rate: 0.15 Elitism: off is reached.

Parameters
populationSizeThe population size.
